# HitchStory Examples

This project contains complete examples for how hitchstory
can be used in four types of projects:

* Command line app
* Python API
* REST API
* Website (with playwright)


## Getting started

$ git clone

$ cd commandline
$ ./key.sh make

## Running all tests

$ ./key.sh regression

## Running one test

$ ./key.sh bdd name

## Running document generation

$ ./key.sh docgen
